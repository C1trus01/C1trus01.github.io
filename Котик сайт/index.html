Добро пожаловать в Cotikbot

Cotikbot — ваш дружелюбный бот для Twitch, который улучшает чат с помощью команд, таких как обновления погоды и интерактивные игры.

  • Ответы ИИ с сарказмом
  • Прогнозы погоды
  • Рекомендации контента
  • Статистика канала

Команды Cotikbot

  • !dip <вопрос> — Получить ответ ИИ на вопрос.
  • !don <вопрос> — Случайный ответ "да/нет".
  • !kto <вопрос> — Случайно выбрать пользователя из чата.
  • !set <город> — Сохранить город для команд погоды.
  • !п [город/ник] — Текущая погода для указанного города или пользователя.
  • !vom — Рекомендация фильма, сериала, мультфильма или аниме.
  • !web <запрос> — Поиск в интернете, возвращает первый результат.
  • !ping — Показать статус бота (время работы, использование ресурсов и т.д.).
  • !шанс <вопрос> — Случайный процент вероятности для вопроса.
  • !сколько — Показать статистику канала (например, количество фоловеров).
  • !up — Показать время работы текущего стрима и количество зрителей.
  • !clear — Очистить память бота (только для админов).
  • !help — Показать список всех доступных команд.
  • !ban <пользователь> [причина] — Забанить пользователя (только для модераторов).
  • !mute <пользователь> <секунд> [причина] — Замутить пользователя на указанное время (только для модераторов).
  • !pred время "вопрос" вариант1/вариант2 — Создать прогноз с вариантами (только для модераторов).
  • !endpred <номер> — Завершить прогноз, выбрав победивший вариант (только для модераторов).

Как добавить Cotikbot на ваш канал

  1. Убедитесь, что вы владелец или модератор канала Twitch.
  2. Свяжитесь с владельцем бота через Telegram: Sosalpososun.
  3. Предоставьте название вашего канала и любую дополнительную информацию, как указано владельцем.
  4. После настройки Cotikbot станет активным в вашем чате!
